# Vertical Swipe Gallery

A mobile-friendly image gallery with vertical swiping between images and horizontal swiping to view different parts of landscape images.

## Features

- Vertical swiping between images
- Horizontal swiping to view left/right halves of landscape images
- Mobile-optimized touch controls
- Clean, minimal design with black background
- Responsive layout that works on all device sizes

## How to Use

1. Swipe up/down to navigate between images
2. On landscape images (images 2-12), swipe left/right to view different halves of the image
3. The first and last images are single-panel and don't support horizontal swiping

## Setup

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/vertical-swipe-gallery.git
   ```
2. Open `index.html` in a web browser

## Adding Images

1. Place your images in the `images/` directory
2. Update the `images` array in `script.js` with your image filenames

## Technologies Used

- HTML5
- CSS3 (Flexbox)
- Vanilla JavaScript (ES6+)
